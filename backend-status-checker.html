❌ Backend Server Không Chạy

Nguyên nhân: Tất cả test og:image thất bại vì backend không chạy.

Giải pháp: Khởi động backend server theo hướng dẫn bên dưới.

🚀 Cách khởi động Backend Server

1 Mở Terminal/Command Prompt mới

Mở terminal riêng biệt (không dùng terminal đang chạy frontend)

2 Di chuyển vào thư mục backend
cd backend
3 Cài đặt dependencies (nếu chưa có)
npm install
4 Khởi động server
npm start

Hoặc nếu npm start không hoạt động:

node server.js
5 Xác nhận server đã chạy

Bạn sẽ thấy trong terminal:

Server running on port 4000
Connected to MySQL database
6 Test lại

  • ✅ og:image sẽ hiển thị ảnh đúng của sản phẩm
  • ✅ Trang /products/7 sẽ dùng ảnh sản phẩm 7
  • ✅ Trang /category/dien-thoai sẽ dùng ảnh category hoặc sản phẩm đầu tiên
